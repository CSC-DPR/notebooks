To be able to run this notebook I recommend the following steps:

1. python3 -m venv new_python_venv   (3.6 <= python_version <= 3.9) ( tested with 3.8)
2. source new_python_venv/bin/activate (different command on Windows)
3. pip install -r requirements.txt
4. jupyter notebook

(NB : the jinja2 template used to display Product Data structure uses css and javascript from cdns => this requires Internet connexion)
